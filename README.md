# 📊 End-to-End Data Pipeline for Sales Analytics

An end-to-end ETL (Extract, Transform, Load) pipeline built with **Python**, **Pandas**, and **MySQL** for processing sales data. The pipeline automates data extraction, transformation, and loading while generating structured datasets for SQL-based analytics and reporting.

## 🚀 Key Features

- Automated ETL workflow
- Extract sales data from CSV files
- Clean and preprocess datasets
- Handle missing values and duplicate records
- Generate derived metrics (Total Revenue)
- Load processed data into MySQL
- Maintain execution logs for pipeline monitoring
- Enable SQL-based sales analytics and reporting

## 🛠️ Tech Stack

- Python
- Pandas
- MySQL
- SQLAlchemy
- PyMySQL
- SQL
